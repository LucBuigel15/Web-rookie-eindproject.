# The Webshop

## Beschrijving

Dit is het eindproject voor Web-Rookie. Het is een eenvoudige webshop gebouwd met HTML, CSS, JavaScript en Bootstrap voor styling en responsief design.

De webshop bevat:

-   Productpagina.
-   Winkelwagen functionaliteit.
-   Responsieve layout.
-   Standaard frontend interactiviteit met JavaScript.

1. Clone de repository.
   git clone git@github.com:LucBuigel15/Web-rookie-eindproject..git
2. Open index.html in een webbrowser:
   Dubbelklik op index.html of open via de browser.
   (Geen extra dependencies nodig, Bootstrap wordt via de CDN geladen.)

## Functionaliteit.

-   Producten worden weergeven met naam afbeelding en prijs.
-   Toevoegen/verwijderen producten uit de winkelwagen.
-   Totaalprijs word automatisch berekend.
-   Toevoegen/verweideren van producten op admin pagina.
-   Orders zien op admin pagina.
-   Responsief design voor desktop en mobiel.
